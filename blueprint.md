# My Pay Archive - 게임 개발 계획

## 개요

이 프로젝트는 하늘에서 떨어지는 사과를 바구니로 받는 간단한 웹 게임입니다. 사용자는 PC에서는 키보드 방향키, 모바일에서는 화면 터치를 통해 캐릭터를 좌우로 움직여 사과를 받을 수 있습니다. 받은 사과의 개수가 점수가 됩니다.

## 디자인 및 기능

### 1. 시각적 요소
- **배경**: 움직이는 구름이 있는 하늘과 땅을 표현하는 동적인 배경.
- **캐릭터**: 귀여운 바구니 아이콘.
- **사과**: 하늘에서 떨어지는 사과 이미지.
- **점수판**: 현재 점수를 표시하는 UI.
- **반응형 디자인**: 다양한 화면 크기(데스크톱, 모바일)에 맞게 레이아웃이 조절됩니다.

### 2. 핵심 기능
- **캐릭터 조작**:
    - **PC**: 키보드 왼쪽/오른쪽 방향키로 캐릭터를 좌우로 이동합니다.
    - **모바일**: 화면의 왼쪽/오른쪽을 터치하여 캐릭터를 해당 방향으로 이동합니다.
- **사과 생성 및 낙하**:
    - 일정 시간 간격으로 화면 상단 임의의 위치에서 사과가 생성됩니다.
    - 생성된 사과는 아래로 떨어집니다.
- **충돌 감지**:
    - 캐릭터의 바구니와 사과가 닿으면 충돌로 감지합니다.
    - 충돌 시 점수가 1점 증가하고, 해당 사과는 화면에서 사라집니다.
- **난이도 증가**:
    - 점수가 올라갈수록 사과의 낙하 속도가 빨라집니다.
    - 점수가 올라갈수록 사과의 생성 빈도가 높아집니다.
- **게임 오버 조건**:
    - 사과 3개를 놓치면 게임이 종료됩니다.
    - "게임 오버" 메시지가 표시된 후 3초 뒤에 자동으로 재시작됩니다.
- **게임 루프**:
    - `requestAnimationFrame`을 사용하여 부드러운 애니메이션과 게임 로직을 처리합니다.

## 기술 스택

- **HTML**: 게임의 기본 구조.
- **CSS**: 스타일 및 시각적 표현, 구름 애니메이션.
- **JavaScript**: 게임 로직, 조작, 애니메이션, 난이도 조절, 게임 오버 처리.
